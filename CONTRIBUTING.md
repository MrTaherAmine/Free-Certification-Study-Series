# Contributing

Thank you for considering a contribution to this free certification study
guide series.

## How to suggest a correction
Open an issue using the "Correction" template if you spot a factual error,
an outdated exam reference, or a broken link.

## How to suggest new content
Open an issue using the "Suggestion" template for new topics, additional
certifications you'd like to see covered, or formatting improvements.

## Scope
This series currently covers PMP®, CCSP®, CC℠, and CISSP®. Suggestions for
future additions (e.g. ISO/IEC 27001 Lead Implementer, ISO/IEC 27005 Risk
Manager) are welcome.
